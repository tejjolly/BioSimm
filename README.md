# BioSimm

This repository is the starting point for BioSimm scripts.

## Post-processing

- `2026-03-10--graphics/post-processing/design_table.py`: generates a study design matrix and a paired metric heatmap.

## Mirror Sync

- Run `scripts/sync_mirror_view.sh` to refresh the local bare mirror, update the browsable mirror clone, and write the latest sync time to `MIRROR_SYNC.txt` in the mirror view.
